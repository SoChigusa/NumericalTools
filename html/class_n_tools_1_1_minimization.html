Detailed Description

Wrapper class for use of Minuit2.

Please define _NTOOLS_USE_MINUIT2 and use compile options below: "ROOTFLAGS = $(shell root-config --cflags)" and "ROOTLIBS = $(shell root-config --libs) -lMinuit2".

Minimize the given function with scanning the initial values.

Initial values are distributed according to the Gaussian distribution with average at arg_vstart and the standard deviation 2 x arg_step.
